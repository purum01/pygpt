모듈 사용방법
파이썬에서 라이브러리를 사용하려면 먼저 import 문을 사용해 해당 라이브러리를 가져와야(import) 합니다. 라이브러리에서 특정 기능이나 모듈만 필요한 경우, 그 부분만 가져올(import) 수도 있습니다.

전체 라이브러리 가져오기

                       
import math
print(math.sqrt(16))  # 16의 제곱근을 출력

특정 함수만 가졍오기

                       
from math import sqrt
print(sqrt(16))  # 16의 제곱근을 출력

라이브러리에 별칭(alias) 사용하기

                       
import math as m
print(m.sqrt(16))  # 16의 제곱근을 출력
