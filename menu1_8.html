파이썬의 dict 타입에 대해 설명해주세요
Python의 dict (딕셔너리) 타입은 키(key)와 값(value)의 쌍을 요소로 갖는 컬렉션입니다. 딕셔너리는 변경 가능(mutable)하고, 순서가 없으며(파이썬 3.7 이상에서는 삽입 순서가 유지되지만, 이는 구현 세부 사항에 따른 것이므로 순서가 중요한 경우에는 OrderedDict을 사용하는 것이 좋습니다), 키는 고유해야 합니다.

주요 특징:

  • 키의 고유성: 딕셔너리에서 키는 고유해야 하며, 중복된 키를 사용하려고 하면 마지막에 할당된 값으로 덮어씌워집니다.
  • 변경 가능한 타입: 딕셔너리는 항목을 추가, 수정, 삭제할 수 있는 변경 가능한(mutable) 타입입니다.
  • 순서: 파이썬 3.7 이상에서는 딕셔너리가 삽입 순서를 유지하지만, 3.6 이하 버전에서는 순서를 유지하지 않습니다. 순서가 중요한 경우 collections.OrderedDict을 사용할 수 있습니다.
  • 다양한 데이터 타입의 키와 값 사용 가능: 딕셔너리의 키로는 변경 불가능한(immutable) 타입만 사용할 수 있으나, 값으로는 어떤 타입도 사용할 수 있습니다.
    •                     
# 중괄호를 사용한 딕셔너리 생성
my_dict = {'name': 'John', 'age': 30}
다음 중 dict 에 관한 특징은 아닌것은 무엇일까요?
  1. Dict의 값의 파이썬의 모든 타입이 사용 가능하다
  2. Dict의 요소들은 수정, 삭제, 추가 작업이 가능하다
  3. Dict의 요소들은 중복되지 않는 키(Key)를 갖는다
  4. Dict의 키(Key)는 파이썬의 모든 타입이 사용 가능하다
