파이썬의 list 객체가 가지는 메소드들에 대해 설명해주세요
파이썬의 list 객체는 순서가 있는 요소의 집합을 다루기 위한 다양한 메소드를 제공합니다. 이러한 메소드들을 활용하여 리스트 내 요소를 추가, 삭제, 검색, 정렬 등 다양한 작업을 수행할 수 있습니다. 아래는 list 객체의 주요 메소드들에 대한 설명입니다:

추가 및 확장

  • append(x): 리스트의 끝에 요소 x를 추가합니다.
  • extend(iterable): 리스트의 끝에 주어진 iterable의 모든 요소를 추가합니다.
  • insert(i, x): 지정된 위치 i에 요소 x를 삽입합니다.

요소 제거

  • remove(x): 리스트에서 값이 x와 같은 첫 번째 항목을 삭제합니다. 해당 요소가 없으면 오류를 발생시킵니다.
  • pop([i]): 지정된 위치 i에 있는 요소를 리스트에서 제거하고, 그 요소를 반환합니다. i가 지정되지 않으면, 리스트의 마지막 요소를 제거하고 반환합니다.
  • clear(): 리스트의 모든 요소를 삭제합니다.

조회 및 검색

  • index(x[, start[, end]]): 리스트에서 x와 값이 같은 첫 번째 항목의 위치를 반환합니다. 해당 요소가 없으면 오류를 발생시킵니다.
  • reverse(): 리스트의 요소 순서를 뒤집습니다.

정렬

  • sort(key=None, reverse=False): 리스트의 요소를 정렬합니다. key 매개변수는 정렬 기준을 지정하며, reverse가 True이면 내림차순으로 정렬합니다.
  • reverse(): 리스트의 요소 순서를 뒤집습니다.

복사

  • copy(): 리스트의 얕은 복사본을 반환합니다.
파이썬의 list 객체의 메소드들을 학습하기 위한 예제코드를 작성해주세요 
                                
# 리스트 생성하기
animals = ['dog', 'cat', 'bird']

# append() 메소드로 리스트 끝에 요소 추가
animals.append('fish')
print("append() 사용 후:", animals)

# insert() 메소드로 특정 인덱스에 요소 추가
animals.insert(1, 'hamster')
print("insert() 사용 후:", animals)

# extend() 메소드로 리스트 확장
more_animals = ['turtle', 'rabbit']
animals.extend(more_animals)
print("extend() 사용 후:", animals)

# 요소 제거하기
animals.remove('cat')  # 첫 번째로 발견된 'cat' 제거
print("remove() 사용 후:", animals)

popped_animal = animals.pop(2)  # 인덱스 2의 요소 제거 및 반환
print("pop() 사용 후:", animals)
print("pop()으로 제거된 요소:", popped_animal)

animals.clear()  # 모든 요소 제거
print("clear() 사용 후:", animals)

# 검색 및 정렬
fruits = ['orange', 'apple', 'banana', 'kiwi', 'apple', 'banana']

# count() 메소드로 요소 개수 찾기
print("apple의 개수:", fruits.count('apple'))

# index() 메소드로 요소 인덱스 찾기
print("banana 위치:", fruits.index('banana'))

# sort() 메소드로 정렬
fruits.sort()
print("sort() 사용 후:", fruits)

# reverse() 메소드로 순서 뒤집기
fruits.reverse()
print("reverse() 사용 후:", fruits)

# 복사
original_list = [1, 2, 3, 4, 5]
copied_list = original_list.copy()
print("원본 리스트:", original_list)
print("복사된 리스트:", copied_list)
