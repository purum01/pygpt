함수 연습문제

  1. 다음은 함수에 대한 설명입니다. 적절하지 않은 것을 고르세요
    • 내장함수는 파이썬에서 기본적으로 제공하는 함수이다.
    • 입력 인자는 함수 실행 후 반환되는 값이다.
    • 내가 원하는 함수를 정의할 수 있다.
    • 함수를 정의하면 여러번 호출할 수 있다.
  2. 다음 코드를 실행하면 오류가 발생합니다. 그 이유를 보기에서 고르세요
    3.                    
def f(a, b): 
c1 = a + b 
c2 = a - b 
return c1, c2 

f(1, 2)
    • 인자의 개수가 잘못되어서
    • 반환값을 잘못 지정해서
    • 함수의 이름이 너무 짧아서
    • 코드의 들여쓰기가 적절하게 되지 않아서
  3. 다음 설명이 맞으면 O, 틀리면 X 표시하세요
    4. 함수에서 입력으로 사용되는 것은 반환값이고, 출력으로 사용되는 것은 인자이다

  4. 다음은 원의 면적을 구하는 코드입니다. 코드의 실행 결과가 None이라고 출력되는데 그 이유를 보기에서 고르세요
    5.                
def area(r) :
    result = 3.14 * r * r 

print(area(3))
    • 함수를 호출하는 코드가 누락되어서
    • return 문이 누락되어서
    • 계산식이 잘못되어서
    • 들여쓰기가 잘못되어서

  8. 다음 코드를 실행하면 오류가 발생합니다. 그 이유를 보기에서 고르세요
    9.                
def wordlist(a, b, c) :
    for item in [a, b, c] :
        print(item)

wordlist(['사과','오렌지','포도'])
    • 함수 호출시 사용한 인자의 개수와 함수 정의시 사용한 인자의 개수가 달라서
    • item 변수를 정의하지 않아서
    • 값을 반환하는 return 코드가 없어서
    • 들여쓰기가 잘못되어서
