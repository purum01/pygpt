name = input("이름을 입력하세요: ")
age = input("나이를 입력하세요: ")
hobby = input("취미를 입력하세요: ")

print("\n[프로필]")
print(f"이름: {name}")
print(f"나이: {age}")
print(f"취미: {hobby}")

퀴즈 게임 제작 

score = 0

print("퀴즈를 시작합니다!")

# 문제 1
answer = input("파이썬의 창시자는 누구일까요? ")
if answer == "귀도 반 로썸":
    print("정답!")
    score += 1
else:
    print("틀렸습니다.")

# 문제 2
answer = input("파이썬의 가장 최신 버전은 무엇일까요? (버전 번호만 입력하세요) ")
if answer == "3.10":
    print("정답!")
    score += 1
else:
    print("틀렸습니다.")

print(f"\n총 점수: {score}점")
  • ❓질문1 : 문제를 더 추가하고 싶어요. 문제 개수가 많아도 깔끔하게 처리하려면 어떻게 할까요?
  • ❓질문2 : 정답을 틀렸을 때 힌트를 주는 기능을 추가하려면 어떻게 하면 좋을까요?
  • 🤔 실습 : 다음 기능을 추가하세요
    1. 점수에 따라 "A등급", "B등급" 등 성적을 매기는 기능 구현
    2. 여러 문제를 반복하여 출제하는 기능 추가
