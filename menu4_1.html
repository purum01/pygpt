형변환 함수

int(), float(), bool(), str(), list(), tuple(), set(), dict() 함수에 대해 설명해주세요
int( )
  • - 지정된 값을 정수(int)로 변환합니다.
  • - 소수점 이하는 버립니다. 문자열이 숫자 형태일 때도 정수로 변환할 수 있습니다.
  • - 예: int(3.5)는 3을 반환합니다.
float( )
  • - 지정된 값을 부동소수점 수(float)로 변환합니다.
  • - 문자열이 숫자 형태일 때도 부동소수점 수로 변환할 수 있습니다.
  • - 예: float("3.5")는 3.5를 반환합니다.
bool( )
  • - 주어진 값의 불리언(bool) 값을 반환합니다.
  • - 대부분의 값은 True로 평가되나, None, 0, 빈 시퀀스([], (), ''), 빈 매핑({}) 등은 False로 평가됩니다.
  • - 예: bool(0)는 False를 반환합니다.
str( )
  • - 객체를 문자열(str) 표현으로 변환합니다.
  • - 숫자, 리스트, 튜플 등 거의 모든 것을 문자열로 변환할 수 있습니다.
  • - 예: str(123)는 "123"을 반환합니다.
list( )
  • - 반복 가능한(iterable) 객체를 리스트(list)로 변환합니다.
  • - 문자열을 리스트로 변환할 때는 각 문자가 리스트의 요소가 됩니다.
  • - 예: list("abc")는 ['a', 'b', 'c']를 반환합니다.
tuple( )
  • - 반복 가능한 객체를 튜플(tuple)로 변환합니다.
  • - 리스트와 유사하게 작동하지만, 반환된 튜플은 변경할 수 없습니다(immutable).
  • - 예: tuple([1, 2, 3])는 (1, 2, 3)을 반환합니다.
set( )
  • - 반복 가능한 객체를 집합(set)으로 변환합니다.
  • - 집합은 중복을 허용하지 않으멀로 중복된 요소는 자동으로 제거됩니다.
  • - 예: set([1, 2, 2, 3])는 {1, 2, 3}을 반환합니다.
dict( )
  • - 키-값 쌍의 시퀀스를 딕셔너리(dict)로 변환합니다.
  • - 또한, 키워드 인자를 사용하여 딕셔너리를 생성할 수 있습니다.
  • - 예: dict([(1, 'one'), (2, 'two')])는 {1: 'one', 2: 'two'}를 반환합니다.

# 숫자를 불리언으로 변환: 0이면 False, 나머지는 True
int_to_bool = bool(0)
print(int_to_bool)  # 출력: False
int_to_bool = bool(1)
print(int_to_bool)  # 출력: True

# 부동소수점 수도 동일한 규칙을 따릅니다.
float_to_bool = bool(0.0)
print(float_to_bool)  # 출력: False
float_to_bool = bool(3.14)
print(float_to_bool)  # 출력: True

# 빈 문자열은 False, 그 외 문자열은 True
str_to_bool = bool("")
print(str_to_bool)  # 출력: False
str_to_bool = bool("Python")
print(str_to_bool)  # 출력: True

# 빈 리스트, 튜플, 집합, 딕셔너리는 False, 요소가 있는 경우 True
list_to_bool = bool([])
print(list_to_bool)  # 출력: False
list_to_bool = bool([1, 2, 3])
print(list_to_bool)  # 출력: True

tuple_to_bool = bool(())
print(tuple_to_bool)  # 출력: False
tuple_to_bool = bool((1, 2, 3))
print(tuple_to_bool)  # 출력: True

set_to_bool = bool(set())
print(set_to_bool)  # 출력: False
set_to_bool = bool({1, 2, 3})
print(set_to_bool)  # 출력: True

dict_to_bool = bool({})
print(dict_to_bool)  # 출력: False
dict_to_bool = bool({1: "one", 2: "two"})
print(dict_to_bool)  # 출력: True

# None은 False로 변환됩니다.
none_to_bool = bool(None)
print(none_to_bool)  # 출력: False

set()을 활용한 예제코드를 생성해주세요 
                
# 리스트에서 중복을 제거하여 집합 생성
list_to_set = set([1, 2, 2, 3, 4, 4, 5])
print(list_to_set)  # 출력: {1, 2, 3, 4, 5}

# 문자열에서 고유 문자를 찾아 집합 생성
str_to_set = set("hello")
print(str_to_set)  # 출력은 문자의 순서에 따라 달라질 수 있음. 예: {'e', 'h', 'l', 'o'}

# 튜플에서 집합 생성
tuple_to_set = set((6, 7, 7, 8))
print(tuple_to_set)  # 출력: {6, 7, 8}

# 빈 집합 생성
# 주의: 빈 집합을 만들기 위해서는 set() 함수를 사용해야 합니다. {}는 빈 딕셔너리를 의미합니다.
empty_set = set()
print(empty_set)  # 출력: set()
dict()을 활용한 예제코드를 생성해주세요 
                
# 키-값 쌍의 시퀀스로부터 딕셔너리 생성
list_of_tuples = [(1, 'one'), (2, 'two'), (3, 'three')]
dict_from_tuples = dict(list_of_tuples)
print(dict_from_tuples)  # 출력: {1: 'one', 2: 'two', 3: 'three'}

# 키워드 인자를 사용하여 딕셔너리 생성
dict_from_kwargs = dict(name="Alice", age=25, city="New York")
print(dict_from_kwargs)  # 출력: {'name': 'Alice', 'age': 25, 'city': 'New York'}

# 빈 딕셔너리 생성
empty_dict = dict()
print(empty_dict)  # 출력: {}

# 리스트의 중첩된 리스트를 사용하여 딕셔너리 생성
list_of_lists = [['name', 'Bob'], ['age', 30], ['city', 'Chicago']]
dict_from_lists = dict(list_of_lists)
print(dict_from_lists)  # 출력: {'name': 'Bob', 'age': 30, 'city': 'Chicago'}

# 튜플의 리스트를 이용하여 딕셔너리의 생성 (키-값 쌍)
tuple_pairs = (('key1', 'value1'), ('key2', 'value2'))
dict_from_tuple_pairs = dict(tuple_pairs)
print(dict_from_tuple_pairs)  # 출력: {'key1': 'value1', 'key2': 'value2'}

# zip() 함수와 리스트를 이용하여 딕셔너리 생성
keys = ['a', 'b', 'c']
values = [1, 2, 3]
dict_from_zip = dict(zip(keys, values))
print(dict_from_zip)  # 출력: {'a': 1, 'b': 2, 'c': 3}
