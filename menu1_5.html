파이썬의 list 타입에 대해 설명해주세요
파이썬의 list 타입은 여러 값을 순서대로 저장하는데 사용되는 데이터 구조입니다. 리스트는 대괄호 []로 표시되며, 항목들은 쉼표로 구분됩니다. 리스트는 다양한 타입의 항목을 포함할 수 있으며, 같은 리스트 내에서도 숫자, 문자열, 심지어 다른 리스트까지 다양한 데이터 타입의 항목들을 함께 저장할 수 있습니다.

주요 특징:

  • 순서가 있음: 리스트의 항목들은 추가된 순서를 유지합니다. 이를 통해 인덱스를 사용하여 특정 항목에 접근할 수 있습니다.
  • 변경 가능: 리스트는 변경 가능한(mutable) 데이터 타입입니다. 즉, 리스트가 생성된 후에도 항목을 추가, 삭제, 변경할 수 있습니다.
  • 다양한 데이터 타입 지원: 리스트는 정수, 부동소수점 수, 문자열 등 포함하여 어떤 타입의 데이터도 저장할 수 있습니다. 또한, 리스트 안에 또 다른 리스트(중첩 리스트)를 포함할 수도 있습니다.
                    
    # 빈 리스트 생성
    my_list = []
    print(my_list)  # 출력: []

    # 여러 데이터 타입을 포함하는 리스트 생성
    my_list = [1, "Hello", 3.14]
    print(my_list)  # 출력: [1, 'Hello', 3.14]

    # 리스트 생성과 동시에 초기화
    another_list = ["apple", "banana", "cherry"]
    print(another_list)  # 출력: ['apple', 'banana', 'cherry']
다음 중 list 에 관한 특징은 아닌것은 무엇일까요?
  1. 리스트의 요소들은 순서가 유지된다
  2. 리스트의 요소들은 수정, 삭제, 추가 작업이 가능하다
  3. 리스트의 요소들은 index를 갖는다
  4. 리스트의 요소들은 동일한 데이터 타입만 가능하다
