tuple Iterable

파이썬의 tuple 객체가 가지는 메소드들에 대해 설명해주세요
파이썬에서 tuple은 변경 불가능한(immutable) 시퀀스 타입입니다. 이 변경 불가능한 특성 때문에, tuple은 리스트(list)에 비해 상대적으로 제한된 수의 메소드만을 가집니다. 주로 tuple 객체가 가지는 메소드는 요소를 조회하거나 요소가 tuple 내에 몇 번 등장하는지 세는 데 사용됩니다. 아래는 tuple 객체의 주요 메소드들에 대한 설명입니다:

count(x)

  • tuple 내에 있는 x 값의 개수를 반환합니다.
  • 예: t = (1, 2, 3, 2, 4)에서 t.count(2)는 2를 반환합니다.

index(x[, start[, end]])

  • tuple 내에서 x 값이 처음으로 나타나는 위치의 인덱스를 반환합니다. x가 tuple 내에 존재하지 않으면 ValueError를 일으킵니다.
  • 선택적인 start와 end 인덱스를 지정하여 검색 범위를 제한할 수 있습니다.
  • 예: t = (1, 2, 3, 2, 4)에서 t.index(3)은 2를 반환합니다.
파이썬의 tuple 객체의 메소드들을 학습하기 위한 예제코드를 작성해주세요 
                                
# tuple 생성
animals = ('dog', 'cat', 'bird', 'dog', 'dog', 'cat')

# 'dog'의 개수를 세기
dog_count = animals.count('dog')
print("'dog'의 개수:", dog_count)

# 'cat'의 개수를 세기
cat_count = animals.count('cat')
print("'cat'의 개수:", cat_count)

# tuple 생성
fruits = ('apple', 'banana', 'cherry', 'orange', 'apple', 'banana')

# 'cherry'의 인덱스 찾기
cherry_index = fruits.index('cherry')
print("'cherry'의 인덱스:", cherry_index)

# 'apple'의 인덱스 찾기 (첫 번째 'apple')
apple_index = fruits.index('apple')
print("첫 번째 'apple'의 인덱스:", apple_index)

# 'apple'의 인덱스 찾기 (두 번째 'apple')
second_apple_index = fruits.index('apple', apple_index + 1)
print("두 번째 'apple'의 인덱스:", second_apple_index)
영화 등장 횟수 세기 
                                
# 영화 캐릭터 tuple
characters = ('Spider-Man', 'Iron Man', 'Captain America', 'Spider-Man', 'Hulk', 'Iron Man')

# 'Spider-Man'의 등장 횟수 세기
spiderman_count = characters.count('Spider-Man')
print("Spider-Man의 등장 횟수:", spiderman_count)

# 'Iron Man'의 등장 횟수 세기
ironman_count = characters.count('Iron Man')
print("Iron Man의 등장 횟수:", ironman_count)
내 생일 찾기 
                
# 가족 생일 tuple
birthdays = ('1월 20일', '4월 5일', '7월 8일', '10월 12일')

# '4월 5일'의 인덱스 찾기
mom_birthday_index = birthdays.index('4월 5일')
print("엄마의 생일은", mom_birthday_index + 1, "번째입니다.")

# '10월 12일'의 인덱스 찾기
my_birthday_index = birthdays.index('10월 12일')
print("내 생일은", my_birthday_index + 1, "번째입니다.")
