포함 연산자

in 과 not in 연산자에 대해 설명해주세요
파이썬에서 in과 not in 연산자는 멤버십 연산자로 분류됩니다. 이 연산자들은 어떤 값이 시퀀스(예: 문자열, 리스트, 튜플, 집합 등) 내에 존재하는지 여부를 검사하는 데 사용됩니다. 결과는 불리언 값인 True 또는 False로 반환됩니다.
in 연산자
  • - in 연산자는 지정된 값이 시퀀스 내에 있을 경우 True를, 그렇지 않으면 False를 반환합니다.
  • - 예제: "a" in "apple"은 True를 반환합니다.
not in 연산자
  • - not in 연산자는 지정된 값이 시퀀스 내에 없을 경우 True를, 존재할 경우 False를 반환합니다.
  • - "b" not in "apple"은 True를 반환합니다.

# 문자열에서의 사용
print('p' in 'python')      # 출력: True
print('z' in 'python')      # 출력: False

# 리스트에서의 사용
my_list = [1, 2, 3, 4, 5]
print(3 in my_list)         # 출력: True
print(6 in my_list)         # 출력: False

# 리스트에서 not in 사용
print(3 not in my_list)     # 출력: False
print(6 not in my_list)     # 출력: True

# 튜플에서의 사용
my_tuple = ('a', 'b', 'c')
print('a' in my_tuple)      # 출력: True
print('d' in my_tuple)      # 출력: False

# 딕셔너리에서의 사용
# 딕셔너리에서 in/not in은 키를 기준으로 검사합니다.
my_dict = {'name': 'Alice', 'age': 25}
print('name' in my_dict)    # 출력: True
print('Alice' in my_dict)   # 출력: False, 'Alice'는 값이며 키가 아님
