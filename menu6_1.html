str Iterable

파이썬의 str 객체가 가지는 메소드들에 대해 설명해주세요
파이썬의 str 객체는 문자열을 다루기 위한 다양한 메소드들을 제공합니다. 이 메소드들은 문자열 검색, 변환, 검증 등을 포함한 광범위한 작업을 수행할 수 있도록 도와줍니다. 아래는 str 객체의 주요 메소드들에 대한 설명입니다:

변환과 조작

  • capitalize(): 문자열의 첫 글자를 대문자로 변환하고, 나머지 글자는 소문자로 변환합니다.
  • upper(): 문자열의 모든 문자를 대문자로 변환합니다.
  • lower(): 문자열의 모든 문자를 소문자로 변환합니다.
  • title(): 문자열의 각 단어의 첫 글자를 대문자로 변환합니다.
  • swapcase(): 문자열의 대문자는 소문자로, 소문자는 대문자로 변환합니다.

검색과 교체

  • count(sub[, start[, end]]): 부분 문자열 sub의 문자열 내 등장 횟수를 반환합니다.
  • startswith(prefix[, start[, end]]): 문자열이 특정 접두사로 시작하는지 확인합니다.
  • endswith(suffix[, start[, end]]): 문자열이 특정 접미사로 끝나는지 확인합니다.
  • find(sub[, start[, end]]): 부분 문자열 sub이 문자열 내에서 처음으로 발견되는 위치를 반환합니다. 찾지 못하면 -1을 반환합니다.
  • replace(old, new[, count]): 문자열 내의 old 문자열을 new 문자열로 교체합니다. count가 지정되면, 해당 개수만큼의 교체를 제한합니다.

분할과 결합

  • split(sep=None, maxsplit=-1): 문자열을 sep을 기준으로 분할하여 리스트로 반환합니다. maxsplit은 분할 횟수를 제한합니다.
  • rsplit(sep=None, maxsplit=-1): 오른쪽부터 분할을 수행합니다.
  • join(iterable): 주어진 iterable의 항목 사이에 문자열을 삽입하여 결합된 문자열을 반환합니다.

공백 처리

  • strip([chars]): 문자열 양쪽 끝의 공백 또는 지정된 문자들을 제거합니다.
  • lstrip([chars]): 문자열 왼쪽 끝의 공백 또는 지정된 문자들을 제거합니다.
  • rstrip([chars]): 문자열 오른쪽 끝의 공백 또는 지정된 문자들을 제거합니다.

검증

  • isdigit(): 문자열의 모든 문자가 숫자인지 확인합니다.
  • isalpha(): 문자열의 모든 문자가 알파벳인지 확인합니다.
  • isalnum(): 문자열의 모든 문자가 알파벳이거나 숫자인지 확인합니다.
  • islower(): 문자열의 모든 알파벳 문자가 소문자인지 확인합니다.
  • isupper(): 문자열의 모든 알파벳 문자가 대문자인지 확인합니다.
파이썬의 str 객체의 메소드들을 학습하기 위한 예제코드를 작성해주세요 
                                
# 문자열 변환 및 조작
text = "hello world"
print(text.capitalize())  # 'Hello world'
print(text.upper())  # 'HELLO WORLD'
print(text.title())  # 'Hello World'
print("PYTHON".lower())  # 'python'
print("AbCdE".swapcase())  # 'aBcDe'

# 문자열 검색 및 교체
message = "Hello, welcome to the jungle"
print(message.count('e'))  # 3
print(message.startswith('Hello'))  # True
print(message.endswith('jungle'))  # True
print(message.find('welcome'))  # 7
print(message.replace('jungle', 'world'))  # 'Hello, welcome to the world'

# 문자열 분할 및 결합
data = "apple,banana,cherry"
print(data.split(','))  # ['apple', 'banana', 'cherry']
print(' '.join(['Join', 'us', 'now']))  # 'Join us now'

# 공백 처리
sample = "   Python   "
print(sample.strip())  # 'Python'
print(sample.lstrip())  # 'Python   '
print(sample.rstrip())  # '   Python'

# 문자열 검증
num_str = "12345"
print(num_str.isdigit())  # True

alpha_str = "Python"
print(alpha_str.isalpha())  # True

alnum_str = "Python3"
print(alnum_str.isalnum())  # True

space_str = "   "
print(space_str.isspace())  # True

lower_str = "python"
print(lower_str.islower())  # True

upper_str = "PYTHON"
print(upper_str.isupper())  # True

알파벳, 숫자, 스페이스의 처리 
                
"""
문자열을 조사하여서 알파벳 문자의 개수, 숫자의 개수, 스페이스의 개수를 출력하는 프로그램
"""
statement = input('문자열을 입력하시오:')
#statement = 'Meav-01-I Dreamt I Dwelt In Marble Halls-192k.mp3'
alphas = 0
digits = 0
space = 0

for c in statement:
    if c.isalpha():
        alphas += 1
    if c.isdigit():
        digits += 1
    if c.isspace():
        space += 1

print('알파벳 문자의 개수 =', alphas)  #알파벳 문자의 개수 = 33
print('숫자 문자의 개수 =', digits)    #숫자 문자의 개수 = 6
print('스페이스 문자의 개수 =', space) #스페이스 문자의 개수 = 6
