for문

파이썬 for문에 대해 설명해주세요
파이썬의 for 문은 시퀀스(리스트, 튜플, 문자열 등) 또는 그 외 반복 가능한(iterable) 객체를 순회하면서, 순회하는 각 항목에 대해 블록 내의 코드를 실행하는 반복문입니다. 파이썬의 for 문은 다른 언어의 for 문과 비교했을 때, 매우 유연하고 읽기 쉬운 구조를 가지고 있습니다.

기본 구조 :

  • 변수: 시퀀스의 각 요소를 참조하는데 사용됩니다. 반복마다 시퀀스의 다음 요소가 이 변수에 할당됩니다.
  • 시퀀스: 리스트, 튜플, 문자열, 또는 반복 가능한(iterable) 다른 객체가 올 수 있습니다.

예제 : 

                            
    # 리스트를 순회하는 예제
    fruits = ['apple', 'banana', 'cherry']
    for fruit in fruits:
        print(fruit)
    
    # range() 함수를 사용한 반복 예제
    for i in range(5):
        print(i)
    
    # 문자열을 순회하는 예제
    for char in "Hello":
        print(char)
구구단 출력하기 
                
    # 3단 출력하기
    dan = 3
    for i in range(1, 10):
        print(f"{dan} * {i} = {dan * i}")
별(*)로 삼각형 그리기 
                
    # 별로 삼각형 그리기
    for i in range(1, 6):
        print('*' * i)
숫자 맞추기 게임 
                
    secret_number = 5
    print("1부터 10까지의 숫자 중 하나를 맞춰보세요.")
    
    for i in range(1, 5):  # 사용자에게 4번의 기회를 줍니다.
        guess = int(input(f"{i}번째 시도: "))
        if guess == secret_number:
            print("정답입니다!")
            break
        else:
            print("틀렸습니다.")
