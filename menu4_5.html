range 내장함수

range( ) 함수에 대해 설명해주세요
range() 함수는 파이썬에서 주어진 범위의 숫자 시퀀스를 생성하는 데 사용됩니다. 이 함수는 for 루프에서 반복을 위한 숫자 시퀀스를 생성하거나, 리스트, 튜플과 같은 다른 데이터 타입으로 변환하는 데 자주 사용됩니다.

기본 사용법 :

  1. 한 개의 인자와 함께 사용: range(stop)
    0부터 stop-1까지의 숫자를 포함하는 시퀀스를 생성합니다. 여기서 stop은 반드시 지정해야 하며, 이 값은 포함되지 않습니다.
  2. 두 개의 인자와 함께 사용: range(start, stop)
    start에서 stop-1까지의 숫자를 포함하는 시퀀스를 생성합니다. 여기서 start 값은 포함되고, stop 값은 포함되지 않습니다.
  3. 세 개의 인자와 함께 사용: range(start, stop, step)
    start에서 stop-1까지, step만큼의 간격으로 숫자를 포함하는 시퀀스를 생성합니다. step은 숫자 간의 간격을 지정합니다.
range()을 활용하여 list 객체를 생성하는 예제를 작성해주세요 
                
# 0부터 4까지의 숫자를 포함하는 리스트 생성
list_example_1 = list(range(5))
print(list_example_1)  # 출력: [0, 1, 2, 3, 4]

# 3부터 6까지의 숫자를 포함하는 리스트 생성
list_example_2 = list(range(3, 7))
print(list_example_2)  # 출력: [3, 4, 5, 6]

# 0부터 9까지 2씩 증가하는 숫자를 포함하는 리스트 생성
list_example_3 = list(range(0, 10, 2))
print(list_example_3)  # 출력: [0, 2, 4, 6, 8]

# 음수 step을 사용하여 10부터 5까지 감소하는 숫자를 포함하는 리스트 생성
list_example_4 = list(range(10, 4, -1))
print(list_example_4)  # 출력: [10, 9, 8, 7, 6, 5]

# 5부터 시작하여 0까지 역순으로 숫자를 포함하는 리스트 생성
list_example_5 = list(range(5, -1, -1))
print(list_example_5)  # 출력: [5, 4, 3, 2, 1, 0]
