dict Iterable

파이썬의 dict 객체가 가지는 메소드들에 대해 설명해주세요
파이썬의 dict (사전) 객체는 키(key)와 값(value) 쌍을 요소로 가지는 컬렉션입니다. dict는 데이터를 효율적으로 저장하고 검색할 수 있도록 해줍니다. 파이썬 사전에는 데이터를 조작하고 접근하기 위한 다양한 메소드가 있습니다. 여기에는 dict 객체의 주요 메소드들에 대한 설명이 포함됩니다:

요소 추가 및 제거

  • clear(): 사전의 모든 항목을 제거합니다.
  • pop(key[, default]): 사전에서 키에 해당하는 항목을 제거하고, 그 값을 반환합니다. 키가 사전에 없는 경우 default를 반환합니다. default가 제공되지 않으면 KeyError가 발생합니다.
  • popitem(): 사전에서 임의의 항목을 제거하고, (키, 값) 쌍으로 반환합니다.
  • update([other]): 사전에 다른 사전 또는 키/값 쌍의 반복 가능한 객체의 내용을 추가합니다.

요소 조회 및 검색

  • get(key[, default]): 사전에서 키에 해당하는 값을 반환합니다. 키가 사전에 없는 경우 default를 반환합니다.
  • keys(): 사전의 모든 키를 담은 뷰(view)를 반환합니다.
  • values(): 사전의 모든 값을 담은 뷰를 반환합니다.
  • items(): 사전의 모든 키/값 쌍을 담은 뷰를 반환합니다.

복사

  • copy(): 사전의 얕은 복사본을 반환합니다.

기타

  • setdefault(key[, default]): 사전에 키가 존재하면 그 값을 반환합니다. 키가 없는 경우, 키와 default 값을 사전에 추가하고 default를 반환합니다. default가 제공되지 않으면 None을 사용합니다.
  • fromkeys(seq[, value]): 시퀀스 seq의 각 요소를 키로 하고, 모두 같은 값 value를 갖는 새로운 사전을 반환합니다.
파이썬의 dict 객체의 메소드들을 학습하기 위한 예제코드를 작성해주세요 
                                
# 사전 생성
person = {'name': 'Alice', 'age': 25}

# 요소 추가
person['city'] = 'New York'
print("요소 추가 후:", person)

# 요소 조회
print("이름:", person.get('name'))
print("나이:", person['age'])

# 요소 제거
removed_city = person.pop('city')
print("제거된 도시:", removed_city)
print("pop() 사용 후:", person)

# 임의의 항목 제거
key, value = person.popitem()
print(f"popitem()으로 제거된 항목: {key} = {value}")
print("popitem() 사용 후:", person)

# 모든 요소 제거
person.clear()
print("모든 요소 제거 후:", person)

# 사전 업데이트
person.update({'name': 'Bob', 'age': 30, 'city': 'Paris'})
print("사전 업데이트 후:", person)

# 키, 값, 항목 순회
print("키 목록:", list(person.keys()))
print("값 목록:", list(person.values()))
print("항목 목록:", list(person.items()))

# 사전 복사
person_copy = person.copy()
print("사전 복사본:", person_copy)

# setdefault 사용
person.setdefault('hobby', 'reading')
print("setdefault() 사용 후:", person)

# fromkeys() 사용
keys = ['a', 'b', 'c']
default_value = 0
new_dict = dict.fromkeys(keys, default_value)
print("fromkeys()로 생성된 사전:", new_dict)

# 학급 친구들의 좋아하는 색깔
favorite_colors = {
    '영희': '파란색',
    '철수': '녹색',
    '민수': '빨간색'
}

# 새로운 친구의 좋아하는 색깔 추가하기
favorite_colors['지민'] = '노란색'
print("지민이의 좋아하는 색깔 추가:", favorite_colors)

# 철수가 좋아하는 색깔 바꾸기
favorite_colors['철수'] = '보라색'
print("철수의 좋아하는 색깔 변경:", favorite_colors)

# 영희의 좋아하는 색깔 확인하기
print("영희의 좋아하는 색깔:", favorite_colors.get('영희'))

# 과일 가게의 과일 가격
fruit_prices = {
    '사과': 1000,
    '바나나': 500,
    '오렌지': 1500
}

# 가격표 출력
for fruit, price in fruit_prices.items():
    print(f"{fruit} 가격은 {price}원입니다.")

# 특정 과일 가격 확인하기
print("바나나 가격:", fruit_prices.get('바나나', '가격 정보가 없습니다.'))
