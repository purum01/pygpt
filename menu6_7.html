파이썬 dict 객체에서 [key] 문법을 사용하여 요소 추출, 수정, 삭제하는 예제 코드를 작성해주세요

요소 추출하기

                                
# dict 생성
person = {'name': 'Alice', 'age': 25, 'city': 'New York'}

# 키를 사용하여 요소 추출
print(person['name'])  # 출력: Alice
# 키가 존재하지 않으면 KeyError 발생
# print(person['occupation'])  # KeyError: 'occupation'

요소 수정하기

                
# 키를 사용하여 요소 수정
person['age'] = 26
print(person)  # 출력: {'name': 'Alice', 'age': 26, 'city': 'New York'}

# 존재하지 않는 키에 값을 할당하면 새로운 요소가 추가됩니다.
person['occupation'] = 'Engineer'
print(person)  # 출력: {'name': 'Alice', 'age': 26, 'city': 'New York', 'occupation': 'Engineer'}

요소 삭제하기

                
# del 키워드를 사용하여 요소 삭제
del person['city']
print(person)  # 출력: {'name': 'Alice', 'age': 26, 'occupation': 'Engineer'}

# 키가 존재하지 않으면 KeyError 발생
# del person['country']  # KeyError: 'country'
